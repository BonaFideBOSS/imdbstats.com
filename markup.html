Text Formatting
[b]...[/b] Make [b]some text[/b] bold Make some text bold
[i]...[/i] Make [i]some text[/i] italic Make some text italic
[u]...[/u] Underline [u]some text[/u] Underline some text
[url]...[/url] Create a hyperlink: [url]https://www.imdb.com/[/url] Create a hyperlink: https://www.imdb.com/
[url=...]...[/url] Create a hyperlink with custom text: [url=https://www.imdb.com/]IMDb[/url] Create a hyperlink with custom text: IMDb
[link=...]...[/link] Another variable of inserting a hyperlink: [link=https://www.imdb.com/]IMDb[/link] Another variable of inserting a hyperlink: IMDb
